Los tipos de Machine Learning. Existen tres tipos de algoritmos en Machine Learning. Aprendizaje supervisado, aprendizaje no supervisado y aprendizaje reforzado. 
Los algoritmos de aprendizaje supervisado consisten en modelos que tienen una variable respuesta o variable objetivo (variable dependiente) y variables explicativas (variables independientes). La variable respuesta es aquella variable que el algoritmo intenta predecir, mientras que las variables explicativas son aquellas que permiten al algoritmo aprender para lograr identificar aquella predicción. La idea principal de estos algoritmos es que la máquina reciba muchos ejemplos del comportamiento de los datos para que esta logre interpretar después, con datos nuevos, cual es la predicción correcta de acuerdo al aprendizaje que tuvo previamente. Algunos algoritmos de aprendizaje supervisado son: Regresiones, árboles de decisión, bosques aleatorios, Redes Neuronales, SVM, etc.
Los algoritmos de aprendizaje no supervisado son algoritmos que no tienen una variable respuesta, por lo tanto no hay una variable por predecir. Este tipo de algoritmos son usados para ejercicios en donde se desea generar segmentos y clusterizar de manera apropiada los registros. Estos modelos buscan principalmente encontrar diferencias y patrones en los datos. Algunos ejemplos de algoritmos de aprendizaje no supervisado son: Análisis de componentes principales, vecinos más cercanos, etc.
Finalmente, el tercer tipo de algoritmo de machine learning es el aprendizaje reforzado. En este tipo de aprendizaje la máquina es entrenada para tomar decisiones. Esto se logra debido a que la máquina se entrena por sí sola continuamente por medio de ensayo y error, de esta manera mejora su nivel de precisión buscando optimizar la respuesta. Este tipo de aprendizaje es utilizado más que todo en videojuegos y carros autónomos y un ejemplo de estos algoritmos es el proceso de la cadena de Markov.
